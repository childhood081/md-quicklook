# Privacy Policy / 隐私政策

**AI 文档快看** is a lightweight Markdown document viewer, editor, and export tool.

Last updated: 2026-06-13

## English

AI 文档快看 does not collect personal data.

The app does not require an account, does not upload user documents, does not include advertising, and does not use analytics SDKs or tracking SDKs.

Markdown files opened by the user are processed locally on the user's device. Word and Excel exports are generated locally and saved only to locations selected by the user. PDF export uses the operating system print dialog and does not upload document content.

The app may store local preferences, such as language choice and recent file paths, on the user's device. This information is not transmitted to the developer or any third party.

AI 文档快看 does not integrate analytics SDKs, advertising networks, crash reporting services, cloud synchronization services, or external authentication providers.

For App Store privacy labels, all data categories should be marked as **Data Not Collected**.

If the app adds features that affect data handling in the future, this privacy policy will be updated before those features are released.

For questions, please use the support page:

https://github.com/childhood081/md-quicklook/issues

## 简体中文

AI 文档快看不收集个人数据。

本应用不需要账号，不上传用户文档，不包含广告，也不使用分析 SDK 或追踪 SDK。

用户打开的 Markdown 文件仅在用户本地设备上处理。Word 和 Excel 导出文件也在本地生成，并只保存到用户自己选择的位置。PDF 导出使用系统打印对话框，不上传文档内容。

本应用可能会在用户设备本地保存偏好设置，例如界面语言和最近打开文件路径。这些信息不会传输给开发者或任何第三方。

AI 文档快看不集成分析 SDK、广告网络、崩溃报告服务、云同步服务或外部登录服务。

App Store 隐私标签建议所有数据类型均标记为 **不收集数据**。

如果未来新增影响数据处理方式的功能，本隐私政策会在相关功能发布前更新。

如有问题，请通过以下支持页面联系：
