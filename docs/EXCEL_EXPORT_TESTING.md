# Excel Export Testing

本文档用于手动测试 md-quicklook 的 Excel 导出功能完善情况。

## 测试准备

使用 `docs/test-docx-export.md` 或创建测试 Markdown 文件，包含各种表格场景。

## 测试用例

### 1. 无表格文档

**操作**：
1. 打开一个没有表格的 Markdown 文件
2. 点击 `导出 Excel`

**预期**：
- 显示提示："当前 Markdown 文档中没有可导出的表格"
- 不生成空 Excel 文件
- 不弹出保存对话框

### 2. 单表格导出

**操作**：
1. 打开包含一个表格的 Markdown 文件
2. 点击 `导出 Excel`
3. 选择保存路径

**预期**：
- 弹出保存对话框，默认文件名正确
- 生成的 .xlsx 包含一个 sheet
- Sheet 名默认为 "Table 1"（或从表格前标题提取）
- 表头行加粗
- 首行冻结（滚动时表头保持可见）
- 列宽自动调整
- 中文内容不乱码

### 3. 多表格导出（多 sheet）

**操作**：
1. 打开包含多个表格的 Markdown 文件
2. 点击 `导出 Excel`

**预期**：
- 每个表格生成一个独立的 sheet
- Sheet 名不重复
- All table data preserved
- Each sheet has independent freeze and column widths

### 4. Sheet 名称从标题提取

**测试内容**：
```markdown
## 预算表

| 项目 | 金额 |
|------|------|
| 办公用品 | 1200 |
| 差旅费 | 3500 |

## 成员名单

| 姓名 | 角色 |
|------|------|
| 张三 | 开发 |
| 李四 | 测试 |
```

**预期**：
- Sheet 1 名为 "预算表"（而不是 "Table 1"）
- Sheet 2 名为 "成员名单"
- 如果标题包含非法字符，已被清理
- 如果标题清理后为空，回退到 "Table N"

### 5. 数字列识别

**测试内容**：
```markdown
| 项目 | 数量 | 单价 | 总价 |
|------|------|------|------|
| A    | 10   | 5.5  | 55.0 |
| B    | 20   | 3.0  | 60.0 |
| C    | 15   | 4.25 | 63.75 |
```

**预期**：
- 数量、单价、总价列在 Excel 中为数字格式（右对齐）
- 项目列保留为文本（左对齐）
- 可以对数字列进行求和等数学运算
- 表头行仍然是文本

### 6. 中文内容

**测试内容**：
```markdown
| 姓名 | 部门 | 备注 |
|------|------|------|
| 张三 | 研发部 | 试用期 |
| 李四 | 市场部 | 已转正 |
```

**预期**：
- 所有中文正确显示，无乱码
- 中英文混排正常
- 中文标点正常

### 7. 空单元格

**测试内容**：
```markdown
| A | B | C |
|---|---|---|
| 1 |   | 3 |
|   | 2 |   |
```

**预期**：
- 空格在 sheet 中是`undefined`，显示为空单元格
- 不影响表格布局

### 8. 编辑模式修改后导出

**操作**：
1. 切换到编辑模式
2. 修改表格内容（添加行、修改单元格）
3. 点击 `导出 Excel`

**预期**：
- 导出的是最新修改后的内容
- NOT the saved content on disk

### 9. 源码模式修改后导出

**操作**：
1. 切换到源码模式
2. 修改 Markdown 表格语法
3. 点击 `导出 Excel`

**预期**：
- 导出的是最新修改后的内容

### 10. 取消导出

**操作**：
1. 点击 `导出 Excel`
2. 在保存对话框中点击取消

**预期**：
- 不生成文件
- 不显示错误提示
- 编辑器状态不受影响

### 11. 语言切换后的导出提示

**操作**：

1. 切换到中文，导出无表格文档。
2. 切换到 English，导出无表格文档。
3. 分别取消一次保存对话框。

**预期**：

- 中文无表格提示为“当前 Markdown 文档中没有可导出的表格”。
- 英文无表格提示为“No exportable tables found in the current document”。
- 取消导出不显示错误 toast。
- 导出保存对话框标题尽量跟随当前语言。

## Excel Sheet 名限制

- Sheet 名最长 31 个字符
- 不能包含：`[ ] : * ? / \`
- 如果有多个表格使用相同的 sheet 名（如标题相同），自动添加 `_2`, `_3` 后缀

## 回归检查

- Word 导出不受 Excel 导出改进影响
- 保存功能不受影响
- 菜单和工具栏的导出行为一致
- 编辑模式和源码模式未保存内容都应从 `currentContent` 导出，而不是从磁盘旧内容导出
