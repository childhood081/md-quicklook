# AI 文档快看导出测试

本文件包含所有常见 Markdown 元素的中英文混排示例，用于验证 Word、PDF 导出与阅读模式的一致性。

## 中文正文测试

AI 文档快看是一个轻量、离线、原生桌面的文档查看与导出工具，专为 AI 生成的 Markdown 文档优化。打开任何 `.md` 或 `.markdown` 文件 — 完全离线，无需账号。

AI 产出的文档通常包含技术分析、数据表格、代码示例等内容。在导出时，应该保持排版风格与阅读模式尽量一致。

### 英文正文测试

This is a paragraph of English text for testing visual consistency across exports. The reading mode should serve as the single source of truth for typography, spacing, and color choices.

Markdown documents generated by AI often include a mix of technical specifications, code examples, data tables, and explanatory text — all of which should look consistent when exported to Word or PDF.

#### 中英文混排测试

在 AI 生成的 Markdown 文档中，中英文混排非常常见。例如 API 文档同时包含中文说明和 English descriptions. 导出时应确保中英文字体分别使用合适的字体 — 中文使用宋体（SimSun）或系统默认中文字体，英文使用 Times New Roman 或 Arial。

**加粗的中文文本** 和 **bold English text** 在导出时应保持加粗样式。

*斜体的中文文本* 和 *italic English text* 在导出时应保持斜体样式。

### JavaScript 代码块

```javascript
function fibonacci(n) {
  if (n <= 1) return n
  let a = 0, b = 1
  for (let i = 2; i <= n; i++) {
    [a, b] = [b, a + b]
  }
  return b
}

console.log(fibonacci(10)) // 55
```

        quantity = item.get("quantity", 1)
        total += price * quantity
