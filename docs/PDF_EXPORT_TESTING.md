# PDF Export Testing / PDF 导出测试清单

## 测试准备

1. 打开 `docs/test-export-visual-consistency.md` 文件
2. 确保阅读模式已加载
3. 对比阅读模式截图和 PDF 输出

## 基础功能测试

- [ ] **工具栏按钮**: 点击工具栏"PDF"按钮，系统打印对话框打开
- [ ] **菜单入口**: 点击"文件 → 导出 PDF…"，系统打印对话框打开
- [ ] **无文件时禁用**: 未打开文件时，工具栏 PDF 按钮不可点击
- [ ] **中英文 UI**: 切换语言后，PDF 按钮文案正确（"PDF"）

## 视觉一致性测试

### 标题
- [ ] H1 标题: 字号、加粗、颜色、下划线（border-bottom）与阅读模式一致
- [ ] H2 标题: 字号、加粗、颜色与阅读模式一致
- [ ] H3 标题: 字号、加粗、颜色与阅读模式一致
- [ ] H4 标题: 字号、颜色、uppercase 与阅读模式一致

### 正文
- [ ] 中文正文: 字体、字号、颜色与阅读模式一致
- [ ] 英文正文: 字体、字号、颜色与阅读模式一致
- [ ] 中英文混排: 字体切换正确
- [ ] 行距: 接近阅读模式（1.6 ~ 1.72）
- [ ] **加粗中文**: 加粗样式保留
- [ ] **bold English**: 加粗样式保留
- [ ] *italic text*: 斜体样式保留
- [ ] [链接]: 颜色与阅读模式一致，打印时显示 URL

### 代码块
- [ ] 代码块背景: 深色（`#2f3b42`）保留
- [ ] 代码块文字: 浅色保留，Shiki 语法高亮颜色可见
- [ ] 等宽字体: Consolas / monospace
- [ ] 长代码行: 自动换行，不被截断
- [ ] 内联代码: 背景和文字色正确

### 表格
- [ ] 表头背景: `#f0f0ed` 保留
- [ ] 表头文字: 颜色和加粗正确
- [ ] 单元格边框: `#d8d8d4` 线条可见
- [ ] 偶数行背景: `#fafaf8` 保留
- [ ] 表格不溢出页面

### 引用块
- [ ] 左边框: `#353183`（indigo）可见
- [ ] 背景色: `#f4f3f0` 保留
- [ ] 文字色: `#2f3b42` 正确

### 分割线
- [ ] 水平分割线显示正常
- [ ] 颜色接近 `#1d1d1b`

### 元素排除
- [ ] 工具栏不出现在 PDF 中
- [ ] 目录面板不出现在 PDF 中
- [ ] 菜单不出现在 PDF 中
- [ ] Front Matter 不出现在 PDF 正文中
- [ ] 空标题不出现在 PDF 中

## 平台特定测试

### macOS
- [ ] 打印对话框中可选择"存储为 PDF"
- [ ] 保存后的 PDF 文件可以用 Preview/Acrobat 打开
- [ ] 中文无乱码

### Windows
- [ ] 打印对话框中可看到"Microsoft Print to PDF"
- [ ] 保存后的 PDF 文件可以正常打开
- [ ] 中文无乱码
- [ ] 如背景色丢失，确认是打印设置问题（非代码问题）

## 已知可接受差异

- 打印时 URL 可能会追加在链接文本后面（CSS `::after` 规则）
- 页边距由打印设置决定，不能精确匹配阅读模式
- 字体渲染由系统字体 + 打印引擎决定，不能保证像素级一致
